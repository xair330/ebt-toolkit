# EBT 胜任力数据分析工具包

## 使用说明

### 1. 准备工作
确保已安装依赖库：
```bash
pip install pandas openpyxl xlrd matplotlib seaborn jieba
```

### 2. 只需改一个文件：`config.py`
| 配置项 | 说明 |
|:------|:----|
| `DATA_FILE` | 新数据源路径（.xls 或 .xlsx）|
| `OUTPUT_DIR` | 分析产出目录（默认与数据同级）|
| `REPORT_PERIOD` | 报告期标识（如 `"2026 Q2"`），出现在标题和图表中 |
| `COMPETENCIES` | 胜任力维度列表（名称、代码需与Excel列名对应）|
| `CATEGORY_KEYS` | 问题大类关键词（可自由增删大类）|
| `THEME` | 配色方案（颜色全部在此修改）|

### 3. 运行
```bash
# 进入项目目录后运行
python run.py
```

### 4. 输出内容
运行后在 `OUTPUT_DIR` 目录下生成：

| 文件 | 内容 |
|:----|:----|
| `图1_问题大类分布.png` | 环形图 + 条形图 |
| `图2_热力矩阵.png` | 胜任力 × 问题大类交叉热力图 |
| `图3_角色对比.png` | 教员/机长/副驾驶失误频次对比 |
| `图4_气泡图.png` | 技术等级 × 胜任力 分布气泡图 |
| `图5_机型雷达图.png` | A330 vs A350 雷达图 |
| `图6_低分频次分布.png` | 各胜任力低分频次条形图 |
| `不足项明细.csv` | 所有低分记录原始明细 |
| `PRO专项分析.csv` | PRO胜任力子类分析 |
| `评语质量排行.csv` | 教员/检查员评语质量得分排名 |
| `分析总报告.md` | Markdown 格式汇总报告 |

## 文件结构
```
ebt_analysis_toolkit/
├── config.py        ← 【唯一需要修改的文件】
├── run.py           ← 主入口，直接运行
├── data_loader.py   ← 数据加载与预处理
├── charts.py        ← 所有图表生成函数
└── README.md        ← 本说明文件
```

## Excel 列结构说明
工具包预设从第 **20列（索引19）** 开始读取胜任力数据，
每项胜任力占 **3列**（优点 / 得分 / 不足）。
若新数据起始列不同，修改 `data_loader.py` 中的 `base_idx = 19` 即可。
